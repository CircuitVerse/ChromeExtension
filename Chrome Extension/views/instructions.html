CircuitVerse Presentation
Embed Extension


An extension that allows the user to embed an interactive
CircuitVerse circuit in Google Slides with a
combination of Google Slides Plugin
and a Chrome Extension

Plugin Installation


Refer to "Plugin Installation" on the README of CircuitVerse-Presentation-Embed : README

Instructions

After installing the plugin (instructions above) to use it :
Click "Embed Circuit" sub-menu on the "CircuitVerse" menu
Enter the circuit's url and click "Embed!"
Now when in presentation mode, users will be able to interact with the image embedded to the presention
